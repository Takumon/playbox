// tag::内部相互参照[]
パラグラフの書き方は <<paragraphs>> を参照してください。

ドキュメントの整理方法は <<section-titles,sections>> を参照してください。
// end::内部相互参照[]


// tag::ドキュメント間の相互参照[]
詳細は <<document-b.adoc#section-b,Section B>> を参照してください。

あなたが、<<document-b#section-b,Section B>> から戻ってきた時に会いましょう。
// end::ドキュメント間の相互参照[]
